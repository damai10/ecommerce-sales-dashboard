# 📊 E-commerce Sales Dashboard

This project analyzes e-commerce sales data using SQL and Power BI, with advanced DAX measures for deeper insights.

## 🔧 Tools Used
- SQL (MySQL)
- Power BI

## 📌 Key Metrics
- Total Sales
- Total Orders
- Total Profit
- Average Order Value (AOV)
- Profit Margin

## 📊 Key Insights
- West region generated the highest sales
- Technology category has the highest revenue contribution
- Sales show a steady increasing trend over months
- Average Order Value provides insight into customer purchasing behavior
- Profit Margin helps evaluate overall business efficiency
